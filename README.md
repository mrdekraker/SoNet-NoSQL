## Description

This is a NoSQL API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This application uses Express.js for routing, a MongoDB database, and the Mongoose ODM.

## Installation

To install the necessary dependencies, run the following command:

```

```

To start the server, run the following command:

